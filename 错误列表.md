# APP 功能模块错误列表

## 发现的问题

### 1. APP 详情页面数据加载问题

**位置**: `/apps/:id/config`

**现象**:
- APP名称显示为空
- APP标识显示为空  
- AppSecret显示为 "-"
- 创建时间显示为 "-"
- 所有统计数据显示为 0

**可能原因**:
- 后端 API `/api/v1/apps/:id` 返回数据格式问题
- 前端数据绑定错误
- appId 参数传递问题

### 2. 审计日志页面无数据

**位置**: `/system/audit`

**现象**:
- 统计数据全部显示为 0
- 表格显示 "No Data"
- 明明数据库中有15条审计日志

**可能原因**:
- 后端 API `/api/v1/audit-logs` 查询条件问题
- 数据表字段映射错误
- 前端数据解析问题
